### 9Ô∏è‚É£ Key UX Decisions
**Decision 1: Recommendation Before Browsing**

*What: Users receive engagement recommendations before seeing profiles*

*Why: Prevents wrong hiring models and reduces cognitive overload*

**Decision 2: City-First Flow**

*What: City selection happens early*

*Why: Builds trust, improves collaboration feasibility, and relevance*

**Decision 3: Cost & Timeline as Ranges**

*What: Display estimated ranges instead of exact values*

*Why: Sets realistic expectations without false precision*
